Aceite o pagamento por pix facilmente com liquidação instantânea.

== Description ==
Nosso plugin WooCommerce possibilita os comerciantes a aceitar facilmente transações instantâneas e seguras através do sistema PIX ou BOLETO. Nosso plugin agiliza o processo de checkout onde tudo funciona de uma maneira automática, permitindo que os clientes façam pagamentos rápidos diretamente de suas contas bancárias, garantindo comodidade, rapidez e confiabilidade para empresas e consumidores.

## *VANTAGENS DO PLUGIN PIX Pay Easy for WOOCOMMERCE*

* Atualização em tempo real após pagamento
* Mais conversão
* Acertos instantanêos via Pix no seu CNPJ
* Integração em 1 clique
* WebHook: Aviso de recebimento em tempo real
* Mais autonomia para cobranças e pagamentos
* Transações ilimitadas
* Sem mensalidades
* Livre de burocracia
* Geração QrCode Pix em Tempo Real
* Pague somente por Pix recebido
* Geração Boleto em Tempo Real
* Pague somente por Boleto Compensado
* Anti-fraude nativo sem custos adicionais
* Painel para acompanhamento de todas as transações

== Installation ==
Para instalar o plugin Pay Easy Para Woocommerce, você precisa de:

* WordPress versão 6.2 ou mais recente (instalado)
* WooCommerce versão 8.0 ou mais recente (instalado e ativado)
* PHP versão 7.2 ou mais recente

= Instruções =

O processo de instalação do plugin é simples e fácil de usar. Após instalar o plugin, os usuários podem acessar facilmente a opção de configurações através do menu principal do administrador. Dentro dessas configurações, os usuários encontrarão um link para criar uma conta na Pay Easy. Assim que a conta for aprovada pelo Pay Easy, os usuários receberão chaves de API de teste e chaves de API de produção. É simples alternar entre os modos de teste e produção.

Além disso, os usuários têm a flexibilidade de modificar o nome e a descrição do gateway de pagamento exibidos na página de checkout. Eles também podem incluir instruções personalizadas que aparecerão na página de confirmação do pedido, orientando os clientes durante o processo de pagamento. Além disso, nas configurações, os usuários podem definir o status do pedido para pagamentos pendentes e pagamentos pagos de acordo com suas preferências.

Além disso, um e-mail de suporte foi incorporado à página de configurações, oferecendo aos usuários um caminho direto para buscar assistência ou esclarecimentos conforme necessário.
